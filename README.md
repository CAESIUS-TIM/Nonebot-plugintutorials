# Nonebot-plugintutorials
Nonebot2的插件编写教程，有些地方可能讲的不好，欢迎大佬们指教！

添加教程可直接 `Fork` 然后进行编辑，提交可直接使用 `Pull Request` 进行提交，并请等待审核及建议。

初学者请按照序号顺序阅读！
